Welcome to TimeGiver's Visualizer

TimeGiver is an open-source, highly-customizable circadian lighting rhythm generator that empowers connected lighting systems to support a healthy cycle of sleep and wakefulness. It takes arguments such as wake time and sunset length and returns an appropriate set of lighting parameters such as brightness and color temperature for the current moment. Rhythms can be generated based on just a few parameters using appropriate defaults or customized extensively according to desired maximums, minimums, rates of change, and slope shapes. TimeGiver also includes a sunlight simulator that uses multiple colored lights to mimic the natural blue-yellow color gradient of sunlight, producing a unique combination of colors for every time of day that recreates everything from the crisp blue of noontime to the golden hues of late afternoon and mellow blues of dusk, all without compromising color rendering. This visualizer beta demonstrates TimeGiver's potential. The height of the graph inside the box represents brightness out of the total possible brightness, and the color of the graph represents the real color or color temperature assigned to the lights at each moment. The following inputs are set to TimeGiver's default settings and can be modified to change the schedule illustration above that is generated live using TimeGiver's actual code. For more information about the following parameters or to learn how to integrate TimeGiver into a project, check out: TimeGiver on GitHub
